# Proyecto Final Frontend

CRUD de estudiantes hecho con React, Vite y React Router.

## Incluye

- Listado, creación, edición y eliminación
- Persistencia en `localStorage`
- Rutas para inicio, listado, detalle y formulario
- Confirmación antes de eliminar
- Estados de carga y error

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Este proyecto funciona en Netlify o Vercel como SPA.

- Build command: `npm run build`
- Output directory: `dist`
